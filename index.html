MKS Coding Challenge

Coding Challenge for application to MakerSquare.

Simulates a simple banking environment,

Can support multiple banks, and multiple users.

Contains three classes:

Person class A new person should have a name set and should have a certain amount of cash that the person keeps with him/her.

Bank class With the bank, you should be able to: * Open a new account using the person's name * Withdraw money from a person's account * Deposit money into a person's account * Transfer money to another bank

Credit_Card class A credit card will have a name, a credit limit (the max balance a card may have), balance informatiion, and an interest rate. A credit card supports basic transactions: * Charges, with support for not exceeding the card's credit limit * Payments, which registers payments to the card's balance * Interest, which applies the interest rate to the card's balance
